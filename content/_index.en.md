## Internship Project

# Smart Attendance SaaS

**Smart Attendance SaaS** is a multi-organization attendance management system built on Amazon Web Services using a **serverless** and **event-driven** architecture.

During the learning phase, I studied foundational AWS services including IAM, EC2, VPC, S3, CloudFront, Route 53, Cognito, API Gateway, Lambda, DynamoDB, SQS, Step Functions, EventBridge, SES, CloudWatch, and Infrastructure as Code tools.

During the project phase, the team analyzed, designed, and implemented a system that supports authentication, clock-in and clock-out, profile management, attendance history, report export, notifications, and operational monitoring.

My main responsibilities focused on **cloud infrastructure, cloud networking, AWS service deployment, security, monitoring, and DevOps support**.
