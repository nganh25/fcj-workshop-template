# Build and Deploy Smart Attendance SaaS on AWS

This workshop explains how to review the source code, run the system locally, deploy the backend with AWS SAM, integrate React with Cognito and API Gateway, test the system, monitor operations, and clean up resources.

## Modules

1. [Architecture and source-code overview](5.1-workshop-overview/)
2. [Prerequisites](5.2-prerequisite/)
3. [Run and deploy the backend](5.3-s3-vpc/)
4. [Integrate the frontend, Cognito, and API](5.4-s3-onprem/)
5. [Testing, security, and monitoring](5.5-policy/)
6. [Resource cleanup](5.6-cleanup/)

## Outcome

After completing the workshop, learners can deploy an MVP using Cognito, API Gateway, Lambda, DynamoDB, S3, and CloudFront, and can inspect CloudWatch logs, IAM permissions, and common deployment errors.
