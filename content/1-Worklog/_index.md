# Nhật ký công việc

## Tổng quan

Quá trình thực tập và thực hiện dự án kéo dài **12 tuần**. Bốn tuần đầu tập trung vào kiến thức AWS nền tảng; các tuần tiếp theo tập trung vào Serverless, phân tích hệ thống, phát triển Smart Attendance SaaS, bảo mật, CI/CD, giám sát và tối ưu.

Vai trò chính của tôi là **Cloud Infrastructure và Cloud Networking**, đồng thời hỗ trợ tích hợp Backend, kiểm thử, bảo mật và triển khai.

## Nội dung theo từng tuần

### [Tuần 1 – Làm quen với AWS và chuẩn bị môi trường](1.1-week1/)

Làm quen với chương trình Workforce Bootcamp – First Cloud AI Journey. Nắm lộ trình, quy định và yêu cầu của chương trình thực tập.

### [Tuần 2 – Dịch vụ Compute và giám sát hệ thống](1.2-week2/)

Tìm hiểu Amazon EC2, EBS và EFS. Thực hành triển khai Web Server trên EC2.

### [Tuần 3 – Hệ thống mạng trên AWS](1.3-week3/)

Tìm hiểu Amazon VPC và thiết kế mạng nhiều lớp. Thực hành Public Subnet, Private Subnet và Route Table.

### [Tuần 4 – Lưu trữ và cơ sở dữ liệu](1.4-week4/)

Tìm hiểu Amazon S3, RDS và DynamoDB. Thực hành Bucket Policy, Versioning và mã hóa S3.

### [Tuần 5 – Kiến trúc Serverless](1.5-week5/)

Tìm hiểu AWS Lambda và Amazon API Gateway. Tìm hiểu Amazon Cognito và JWT.

### [Tuần 6 – Phân tích và thiết kế Smart Attendance SaaS](1.6-week6/)

Xác định bài toán và phạm vi MVP. Phân tích chức năng của Admin, Manager và Employee.

### [Tuần 7 – Frontend và xác thực người dùng](1.7-week7/)

Xây dựng React SPA. Tạo AuthContext quản lý phiên đăng nhập.

### [Tuần 8 – API và chức năng điểm danh](1.8-week8/)

Triển khai API Gateway HTTP API v2. Cấu hình Cognito JWT Authorizer.

### [Tuần 9 – Báo cáo và xử lý sự kiện](1.9-week9/)

Hoàn thiện DynamoDB Single-Table Design. Xây dựng chức năng xuất báo cáo.

### [Tuần 10 – Bảo mật hệ thống](1.10-week10/)

Cấu hình AWS WAF và tìm hiểu AWS Shield. Quản lý bí mật bằng Secrets Manager.

### [Tuần 11 – CI/CD, kiểm thử và giám sát](1.11-week11/)

Hoàn thiện Infrastructure as Code. Xây dựng CodeBuild và CodePipeline.

### [Tuần 12 – Tối ưu và hoàn thiện dự án](1.12-week12/)

Rà soát toàn bộ kiến trúc. Tối ưu hiệu năng và chi phí.

## Kết quả tổng quát

- Hoàn thành kiến thức nền tảng về Compute, Networking, Storage và Database trên AWS.
- Xây dựng MVP Smart Attendance SaaS theo kiến trúc Serverless và Event-Driven.
- Tích hợp Cognito, API Gateway, Lambda, DynamoDB, S3, SQS, Step Functions, EventBridge và SES.
- Áp dụng CloudFormation/SAM, CodeBuild, CodePipeline, CloudWatch và X-Ray.
- Nâng cao kỹ năng GitHub, làm việc nhóm, xử lý lỗi và trình bày giải pháp.
