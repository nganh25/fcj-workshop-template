# Worklog

## Overview

The internship and project work covered **twelve weeks**. The first four weeks focused on AWS foundations. The following weeks covered serverless architecture, system analysis, Smart Attendance SaaS development, security, CI/CD, monitoring, and optimization.

My main role was **cloud infrastructure and cloud networking**, with additional support for backend integration, testing, security, and deployment.

## Weekly contents

### [Week 1 – AWS orientation and environment setup](1.1-week1/)

Become familiar with the Workforce Bootcamp – First Cloud AI Journey program. Understand the internship roadmap, rules, and deliverables.

### [Week 2 – Compute services and monitoring](1.2-week2/)

Study Amazon EC2, EBS, and EFS. Deploy a web server on EC2.

### [Week 3 – AWS networking](1.3-week3/)

Study Amazon VPC and multi-tier network design. Practice public and private subnets and route tables.

### [Week 4 – Storage and databases](1.4-week4/)

Study Amazon S3, RDS, and DynamoDB. Practice S3 bucket policies, versioning, and encryption.

### [Week 5 – Serverless architecture](1.5-week5/)

Study AWS Lambda and Amazon API Gateway. Learn Amazon Cognito and JWT authentication.

### [Week 6 – Smart Attendance SaaS analysis and design](1.6-week6/)

Define the problem and MVP scope. Analyze Admin, Manager, and Employee features.

### [Week 7 – Frontend and user authentication](1.7-week7/)

Build the React SPA. Create an AuthContext for session management.

### [Week 8 – API and attendance functions](1.8-week8/)

Deploy API Gateway HTTP API v2. Configure a Cognito JWT authorizer.

### [Week 9 – Reporting and event processing](1.9-week9/)

Finalize the DynamoDB single-table design. Build report export functions.

### [Week 10 – System security](1.10-week10/)

Configure AWS WAF and study AWS Shield. Manage secrets with Secrets Manager.

### [Week 11 – CI/CD, testing, and monitoring](1.11-week11/)

Finalize Infrastructure as Code. Build CodeBuild and CodePipeline workflows.

### [Week 12 – Optimization and project completion](1.12-week12/)

Review the complete architecture. Optimize performance and cost.

## Overall outcomes

- Built a foundation in AWS compute, networking, storage, and databases.
- Developed a Smart Attendance SaaS MVP using serverless and event-driven architecture.
- Integrated Cognito, API Gateway, Lambda, DynamoDB, S3, SQS, Step Functions, EventBridge, and SES.
- Applied CloudFormation/SAM, CodeBuild, CodePipeline, CloudWatch, and X-Ray.
- Improved GitHub, teamwork, troubleshooting, and technical presentation skills.
