# Feedback and Suggestions

## Overall Assessment

The First Cloud AI Journey program provides a path from AWS foundations to practical project delivery. It is suitable for students interested in cloud, DevOps, or backend engineering.

## What I Appreciated

- Diverse learning materials and technical topics.
- Support from mentors and team members.
- Opportunities for teamwork and real AWS architecture design.
- A project that connects compute, networking, storage, security, and DevOps.
- The worklog website, which improves technical documentation skills.

## Challenges

- A large amount of AWS knowledge in a limited period.
- Some services require more hands-on practice.
- Frontend, backend, IaC, and CI/CD integration can create complex issues.
- AWS cost and IAM permissions require careful control.

## Recommendations

- Provide clear weekly output checklists.
- Add dedicated sessions for IAM, CORS, CloudFormation, and CI/CD debugging.
- Standardize the bilingual worklog template from the beginning.
- Hold a mid-project architecture review before extensive development.
- Provide a sandbox or more detailed cost-control guidance.
- Increase demo and feedback time for each team.

## Future Expectations

I hope the program continues its hands-on approach and adds more content about AI/ML, Kubernetes, Terraform, security, and FinOps. I would recommend the program to students interested in cloud because it offers valuable practical learning and project experience.
